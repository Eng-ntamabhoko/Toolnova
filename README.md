# ToolNova

Online tools platform built with Laravel.

## Requirements
- PHP 8.2+
- Composer
- Node.js
- MySQL

## Installation

Clone project then run:

composer install

Copy env:

cp .env.example .env

Generate key:

php artisan key:generate

Configure database in .env

Run migrations:

php artisan migrate

Run server:

php artisan serve

Open:

http://127.0.0.1:8000
